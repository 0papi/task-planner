# Datasets of a todo

- Title - string
- Content / Body - string
- Description - optional field - string
- Date Created - Date type (auto generated)
- isCompleted - boolean
- isEdited - boolean
- lastEdited - Date type (auto generated)
- priority - 'high'| 'medium' | 'low'
